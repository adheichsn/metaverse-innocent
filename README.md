## Get Started

### create file .env

```
DATABASE_URL=
NEXTAUTH_SECRET=

```

```
npm install
```

```
npx prisma migrate dev --name init
```

```
npx shadcn-ui@latest add toast
```

```
npm run dev
```

npm run build
